# lita-utc

It's important to know what the UTC time is. This way you'll know quickly!

## Installation

Add lita-utc to your Lita instance's Gemfile:

``` ruby
gem "lita-utc"
```

## Configuration

None!

## Usage

```
14:37 <j^2> utc
14:37 <j2bot> The UTC time and date is now: 2015-06-01 22:41:49 UTC
```
